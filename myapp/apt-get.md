# apt-get
Linux下的包管理工具，必需要掌握的linux命令之一。

---

#####运行环境
Linux

#####安装
linux自带

#####使用
1. 安装软件
 ```
 sudo apt-get update
 sudo apt-cache search FORMULA
 sudo apt-get install FORMULA
 ```
 
 2. 卸载软件
```
sudo apt-get uninstall FORMULA
```

3. 更新软件
```
sudo apt-get update
sudo apt-get -u upgrade
```

4. 具体命令使用介绍
```
apt-get help
```
或
```
man apt-get
```
